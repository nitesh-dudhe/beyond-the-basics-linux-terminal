Mastery Commands

cut
Extract specific fields or characters from each line of input.
echo "root:x:0:0:root:/root:/bin/bash" | cut -d':' -f1,7grep `user` /etc/passwd | cut -d':' -f1,6,7
sort
Sort lines of text files alphabetically, numerically, or by field.
ps -ef | sort -k3 -nr | head -5cat dummy.log | cut -d'|' -f2 | sort
uniq
Report or omit repeated lines. Works best after sort.
cat dummy.log | cut -d'|' -f2 | sort | uniq
watch
Run a program repeatedly, displaying output & errors. Great for real-time monitoring.
watch -n 1 -d 'df -h | grep /dev/nvme1n1p2'
tee
Read from stdin and write to stdout and files simultaneously.
./script.sh | tee script_execution.log
which
`which` finds the first match in PATH.
which python3which nginx
whereis
`whereis` finds binary, source, and man pages.
whereis python3
file
Tests each argument and classifies it — never trust file extensions.
file mystery_filefile /path/to/mystery_file
fc
Open last command in $EDITOR, save, and it executes. Never retype long commands.
fc -s python=python3
wc
Print newline, word, and byte counts for files.
wc -l wc -l *.`ext` wc -l /path/to/file
alias
Define shortcuts for commands. Add to ~/.bashrc for persistence.
alias `short_cmd`=`longer_cmd` alias kg="kubectl get"

use like this for terraform:
alias tf="terraform" tf init tf plan tf apply

Pro Tips & Shortcuts

💡 Add aliases permanently
add in /home/`user`/.bashrc file at the EOF
💡 Make default editor as Vim
• add in /home/`user`/.bashrc file at the EOF • export EDITOR=vim • export VISUAL=vim • for opening the cronjobs, fc in the default editor
💡 Speed demon
• Ctrl + r -> Reverse history search • Ctrl + a -> Jump to start • Ctrl + e -> Jump to end • !! -> Re-run last command • Alt + . -> Last argument
💡 Stop Cat Abuse
Abusing cats is a prevalent problem in the Linux user community. Instead of using
cat `filename` | grep `pattern` cat `filename` | head -n 10 cat `filename` | less
use:
grep `pattern` `filename` head -n 10 `filename` less `filename`
💡 Navigation Saviors
• Toggle to last directory.
cd -
• Re-run last command with sudo.
sudo !!
• Learn Vim in 20 mins.
